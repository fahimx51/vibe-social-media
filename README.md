# VIBE
### Your world, your rhythm, your people

VIBE is a modern social media platform built with the MERN stack, designed for real-time interaction and seamless user experience.

🔗 **Live Demo:** [https://vibe-your-platform.web.app/](https://vibe-your-platform.web.app/)

---

## 🚀 Key Features

- **Real-time Interactions:** Instant messaging and live notifications using Socket.io.
- **Dynamic Feed:** Create posts, like, and comment on other people's vibes.
- **24-Hour Stories:** Share moments that automatically disappear after 24 hours using time-filtered database queries.
- **Social Networking:** Full follow/unfollow system to build your community.
- **Media Management:** High-performance image storage and optimization via Cloudinary.
- **Responsive Design:** A mobile-first UI that feels smooth on any device.
- **Secure Authentication:** JWT-based authentication with secure HTTP-only cookies and cross-site (CORS) protection.

---

## 🛠️ Tech Stack

### Client-Side
- **React.js**: Modern UI library for a fast, single-page experience.
- **Tailwind CSS**: Utility-first styling for a clean and responsive layout.
- **Axios**: Promised-based HTTP client for API communication.
- **Socket.io-client**: Enabling real-time, bidirectional communication.

### Server-Side
- **Express.js & Node.js**: Robust backend framework.
- **MongoDB & Mongoose**: Flexible NoSQL database with schema modeling.
- **Cloudinary & Multer**: Streamlined image processing and cloud storage.
- **JWT & Cookies**: Secure user session management.
- **Socket.io**: Powering the real-time notification and chat engines.
